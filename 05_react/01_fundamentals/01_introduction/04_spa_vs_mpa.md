# 🟢 SPA (Single Page Application)

## Definition

A web app that loads one single HTML page and updates content
dynamically without refreshing the whole page.

## How it works

-   Page loads once
-   JavaScript changes content
-   No full page reload

## Examples

-   Facebook
-   Instagram
-   Gmail

These feel smooth because pages don't reload.

------------------------------------------------------------------------

## ✅ Advantages of SPA

-   Faster after first load
-   Smooth user experience
-   Feels like a mobile app
-   Great for dashboards & web apps

## ❌ Disadvantages of SPA

-   First load can be slow
-   SEO can be tricky
-   More complex frontend logic

------------------------------------------------------------------------

# 🔵 MPA (Multi Page Application)

## Definition

A traditional website where each page is a separate HTML file, and every
click loads a new page from the server.

## How it works

-   Click link
-   Server sends new HTML page
-   Browser reloads completely

## Examples

-   Amazon
-   Wikipedia

When you click a link, the page refreshes.

------------------------------------------------------------------------

## ✅ Advantages of MPA

-   Better SEO by default
-   Simpler backend rendering
-   Easier for content-heavy sites

## ❌ Disadvantages of MPA

-   Slower navigation (full reload)
-   Less smooth experience

------------------------------------------------------------------------

# 🧠 Simple Comparison

  ------------------------------------------------------------------------
  Feature        SPA                          MPA
  -------------- ---------------------------- ----------------------------
  Page Reload    ❌ No full reload            ✅ Reload every page

  Speed          Fast after first load        Slower navigation

  SEO            Needs extra setup            Good by default

  Best For       Web apps, dashboards         Blogs, content sites
  ------------------------------------------------------------------------

------------------------------------------------------------------------

# 🎯 For Your Projects

Usually SPA (React frontend + Django backend API)

## 📚 LMS

Definitely SPA --- dashboards, progress updates, live changes.
