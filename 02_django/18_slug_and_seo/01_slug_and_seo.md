## What is a Slug?

A **slug** is a **URL-friendly, human-readable string** used in URLs to identify a resource.

### Example URL

```
/blog/what-is-django-orm/
```

Here:

```
what-is-django-orm  ‚Üê slug
```

## Why Slugs Are Important for SEO

- Readable URLs (users + search engines)
- Keywords in URL improve SEO
- Better click-through rate

## What is `SlugField`?

`SlugField` is a Django model field designed to store slugs.

```python
from django.db import models

class Post(models.Model):
    title = models.CharField(max_length=200)
    slug = models.SlugField()
```

---

## SlugField Properties

- Allows letters, numbers, hyphens, underscores
- Stored as `VARCHAR` in database
- Not auto-generated by default

---

## Common `SlugField` Options

### Unique Slug


## Auto-generate Slug from Title

```python
from django.utils.text import slugify

class Post(models.Model):
    title = models.CharField(max_length=200)
    slug = models.SlugField(unique=True)

    def save(self, *args, **kwargs):
        if not self.slug:
            self.slug = slugify(self.title)
        super().save(*args, **kwargs)
```

---

## Handling Duplicate Slugs

```python
def save(self, *args, **kwargs):
    if not self.slug:
        base_slug = slugify(self.title)
        slug = base_slug
        count = 1

        while Post.objects.filter(slug=slug).exists():
            slug = f"{base_slug}-{count}"
            count += 1

        self.slug = slug
    super().save(*args, **kwargs)
```

## Slug in URLs (SEO-Friendly Routing)

### urls.py

```python
path("blog/<slug:slug>/", views.post_detail, name="post_detail")
```

### views.py

```python
def post_detail(request, slug):
    post = Post.objects.get(slug=slug)
    return render(request, "post_detail.html", {"post": post})
```

## SEO Best Practices for Slugs

- Use lowercase
- Use hyphens (`-`), not underscores
- Keep it short & meaningful
- Avoid stop words (and, the)
- Never change slugs after publishing

---
